# RustCUI-Editor
Web editor for cui of rust game.

# Deps
Need for webpack & webpack-dev-server install as global

`
sudo npm i webpack -g
sudo npm i webpac-dev-server -g
`

# Installation
`npm install`

# Run
`webpack-dev-server`

# Build
`webpack`

And open in browser
`http://localhost:1337/editor`
